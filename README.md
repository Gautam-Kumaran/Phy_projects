# Physics Projects

This repository contains a collection of physics-related coding projects I worked on during my undergraduate degree.

The main topics include:

- Van der Pol oscillators
- Nonlinear dynamics
- Burgers' equation
- Sturm sequences
- Vibrations of a circular membrane
- Numerical simulations and visualisations

Most of the projects are written in Python using libraries such as NumPy, SciPy and Matplotlib.

## Files

Some of the main scripts are:

- `Van-Der-Pol Equation.py`
- `Van-Der-Pol2.py`
- `Van-der-pol3D.py`
- `BurgersEquation.py`
- `Sturm_Sequences.py`
- `Vibrations of circular membrane/Simulations.py`

## Running the Code

Run any script directly with Python. For example:

```bash
python "Van-Der-Pol Equation.py"
```

Some scripts create plots or animations, so they are best run in a local Python environment.

## About

This repository is mainly a record of my progress in computational physics and numerical methods.
